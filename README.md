## 🛠️ 사용 기술 및 라이브러리

- **React** : 컴포넌트 기반 UI 개발
- **React Router** : 페이지 이동 및 동적 라우팅
- **Context API + useReducer** : 전역 상태 관리 및 CRUD
- **localStorage**: 클라이언트 측 데이터 저장
- **CSS Modules**: 컴포넌트별 스타일 적용
- **Vercel** : 정적 웹 애플리케이션 배포
